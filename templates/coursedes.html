What is this class about?

AngularJS is a JavaScript framework created by Google. AngularJS Fundamentals and Practice is a Project based class. It is geared towards those new to AngularJS. The class - presented as a series of 29 videos - is divided into 2 parts.

  • The first part focuses on learning the fundamental concepts of AngularJS via coding screencasts(plunkr), slides and lectures.
  • The second part utilizes the concepts taught to create an AngularJS Application (with a coding screencast)
